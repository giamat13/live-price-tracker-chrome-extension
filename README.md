# Live Price Tracker Chrome Extension 🚀

תוסף לכרום (Manifest V3) המאפשר מעקב בזמן אמת אחרי שינויים באלמנטים ספציפיים בתוך דפי אינטרנט (כמו מחירים, סטטוס מלאי, או כותרות חדשות). התוסף משתמש ב-`MutationObserver` כדי לזהות שינויים בתוך ה-DOM ללא צורך בריענון הדף ושולח התראות ישירות למערכת ההפעלה.

## ✨ תכונות עיקריות
- **מעקב בזמן אמת:** זיהוי מיידי של שינויי טקסט או תוכן בעזרת טכנולוגיית `MutationObserver`.
- **תמיכה ב-Manifest V3:** בנוי לפי התקן המודרני והמאובטח ביותר של Google Chrome.
- **התראות דסקטופ:** קבלת התראה למחשב גם כשהלשונית פועלת ברקע.
- **גמישות מלאה:** אפשרות להז** כל CSS Selector (כגון Class או ID) למעקב מותאם אישית.
- **אופטימיזציה (Debounce):** מנגנון מובנה למניעת הצפת התראות בזמן עדכונים מהירים באתר.

## 🛠 טכנולוגיות
- **JavaScript** (ES6+)
- **Chrome Extension API** (Storage, Notifications, Runtime)
- **HTML/CSS** (ממשק המשתמש של הפופ-אפ)

## 📖 איך זה עובד?
התוסף מאפשר למשתמש להגדיר "מטרה" (Target) בדף האינטרנט. ברגע שהקוד מזהה שינוי במבנה ה-HTML או בטקסט של אותו אלמנט, נשלח סיגנל ל-Service Worker של התוסף שמקפיץ הודעת מערכת למשתמש עם הערך החדש שנמצא.
